Descripción del Proyecto
Este proyecto implementa un modelo de Machine Learning basado en Random Forest para clasificar flores del dataset Iris. Además, se desarrolló un dashboard interactivo en Streamlit que permite explorar los datos y realizar predicciones en tiempo real.

Objetivo
Clasificar automáticamente flores Iris en las especies:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

Dataset Utilizado
Dataset Iris con 150 registros y variables de sépalos y pétalos.

Tecnologías Utilizadas
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Plotly
- Matplotlib
- Seaborn

Metodología Aplicada
1. Data Understanding
2. Exploratory Data Analysis
3. Data Preprocessing
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Dashboard Interactivo
